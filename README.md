OpenCPU App: lwmplot
---------------------

Simple OpenCPU Application. To install in R:

    library(opencpu)
    ocpu_start_app("git-ashish/lwmplot")

Use the same function locally:

    library(lwmplot)
    lwmplot()
    ?lwmplot

For more information about OpenCPU apps, see [opencpu.js](https://www.opencpu.org/apps.html)
